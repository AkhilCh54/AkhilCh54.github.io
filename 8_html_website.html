Stock News (HTML Website)

Built using Python, Flask, HTML, CSS, JS & NEWS API

A stock market, equity market, or share market is a public entity for the trading of company stock and derivatives. Stock exchanges provide facilities for stockbrokers to trade stocks and other securities.

I created an application of it using Python. Here are some highlights of the project:

  • This website presents the latest news about the stock market.
  • It also provides updates about companies like Nokia, Tesla, Intel, Google, and Amazon.
  • This project was built using: Python, HTML, JS, CSS, Flask, Jinja, NEWS API, and OOP.
