Pong Game

The first commercially successful ping pong video game was *Pong*, released in November 1972. Created by Allan Alcorn as a training exercise for Atari, it unexpectedly impressed co-founders Nolan Bushnell and Ted Dabney, who decided to release it. *Pong* became an instant hit and played a key role in launching the video game industry.

I recreated a version of Pong using Python. Here's how the game works:

  • It's a 2-player game that can be played simultaneously.
  • Built using object-oriented programming (OOP) and structured into four main files: main.py, ball.py, scoreboard.py, and paddle.py.
  • main.py integrates all components and controls the game flow.
  • paddle.py creates two paddles on opposite sides of the screen, controlled using the keys ('W', 'S') for the left paddle and ('PgUp', 'PgDn') for the right paddle.
  • ball.py manages the ball, which spawns at the center of the screen and bounces off paddles and walls.
  • scoreboard.py tracks and displays player scores.
  • The first player to reach 10 points wins the game.
