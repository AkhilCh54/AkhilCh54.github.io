Snake Game (OOP)

The Snake game which was first introduced in 1997 by Nokia was a huge success, and it helped to popularize the game around the world. Snake is still one of most popular game today.

I created a version of it using Python. Here are some of the snippets on how the game works.

  • This game is programmed with OOP. It contains four parts : main.py, snake.py, scoreboard.py, and food.py
  • The main.py combines different code snippets and makes sure the game works properly.
  • The snake.py creates the snake and tells it how to move
  • The food.py module creates food at a random location on the map. Once the food is eaten by the snake, a new food item is created at a different random location.
  • The scoreboard.py counts the score.
