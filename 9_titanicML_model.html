Predicting Titanic Passengers' Survival Using Machine Learning

Built a predictive model using feature engineering and ensemble learning to determine which passengers survived the Titanic disaster.

This project builds a machine learning model to predict which passengers survived the sinking of the RMS Titanic, using real-world passenger data from the Kaggle Titanic Competition. It explores how features like age, gender, class, and fare affected survival outcomes, and applies data preprocessing, feature engineering, and classification models to generate predictions on unseen data.

  • Accuracy Achieved: The final model achieved an accuracy of 83% on validation data, indicating strong predictive performance given the simplicity of the dataset.
  • Most Influential Features:
    1. Sex: Female passengers had a significantly higher survival rate.
    2. Pclass: Passengers in 1st class were far more likely to survive than those in 2nd or 3rd class.
    3. Family Size & IsAlone: Passengers traveling alone had lower survival rates, highlighting the impact of group travel on survival chances.
    4. Fare & Embarked: Higher fares and certain embarkation ports correlated with better survival odds, possibly due to socio-economic factors.
  • Feature Engineering Impact: Adding derived features like FamilySize, IsAlone, and Title significantly boosted model performance over raw data alone.

This project helps build intuition around issues of bias and fairness, as it reveals how factors like gender, class, and socio-economic status influenced life-or-death outcomes. Additionally, it serves as a practical introduction to the end-to-end data science workflow, making it an excellent project for learning, portfolios, or interview preparation.
