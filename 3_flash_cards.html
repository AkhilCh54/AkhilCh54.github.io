Flash Cards (GUI)

Flash cards are a powerful tool for language learning as they promote active recall, helping to strengthen long-term memory.

They can be used to practice vocabulary, grammar, and even pronunciation in a focused, efficient way

I built a custom flash card app using Python. Here's how it works:

  • Developed using procedural programming and a graphical user interface (GUI).
  • Implemented with the Tkinter module in Python.
  • Displays a word in Hindi, then automatically flips the card after 3 seconds to reveal the English translation.
