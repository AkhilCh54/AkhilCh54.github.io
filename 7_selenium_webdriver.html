Selenium WebDriver

Selenium WebDriver is an open-source tool that allows you to automate web browsers. It is used for testing web applications, as well as for automating tasks that would otherwise be performed manually.

I created an application using Python. Here are some of the snippets on how it works:

  • The project scrapes data from Zillow.com to generate a Google spreadsheet of homes available for rent/buy.
  • Users only need to input the Zillow URL and desired location.
  • Selenium WebDriver automatically fills out Google Forms with the house data (price, address, availability, etc.).
  • The final output is a simple spreadsheet for quick decision-making.
