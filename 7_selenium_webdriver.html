Selenium WebDriver

Selenium WebDriver is an open-source tool that allows you to automate web browsers. It is used for testing web applications, as well as for automating tasks that would otherwise be performed manually.

I created a application of it using Python. Here are some of the snippets on how the game works.

  • In this Project I use the data scraped from "Zillow.com" website to create a google spreadsheet of the houses available for rent/buy based on the user's selection.
  • The Whole process is automated. The user just has the give the url of "zillow.com" along with the area where he/she wants to rent/buy a house.
  • Selenium webdriver collects the data and prints it into the google forms, which later on gets added to google spreadsheet.
  • From adding the data of house rent per month, address, from when the house is available, etc. the selenium webdriver does everything and presents the data in a simple format for the user to read and make a fast decision.
