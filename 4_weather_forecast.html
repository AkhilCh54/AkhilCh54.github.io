Weather Forecast

A simple weather update project using API and email automation.

A weather forecast is a prediction of atmospheric conditions for a specific place and time, based on data from various meteorological sources.

Here's how I created an automated version using Python:

  • Fetched live weather data using the Weather API.
  • Used Python's smtplib to send formatted email updates.
  • Presented the data in a clean table format for user clarity.
