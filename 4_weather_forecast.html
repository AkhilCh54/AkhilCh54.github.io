Weather Forecast

A weather forecast is a prediction of the weather conditions for a particular place and time. It is made by collecting quantitative data about the current state of the atmosphere, land, and ocean and using meteorology to project how the atmosphere will change at a given place.

I created a application of it using Python. Here are some of the snippets on how the game works.

  • In this Project I use the data scraped from "Weather API" website to send an email to user about today's weather forecast.
  • This Project is done using API and smtplib in python
  • The API gets data from the website and sends it to user through the email
  • The data is in the form of a table, so that the users can understand it easily.
