Web Scraping

Web scraping is the process of extracting data from websites. It is a type of data mining that uses automated tools to extract data from websites.

I created an application using Python. Here's how it works:

  • Scrapes data from the Billboard Hot 100 to create a Spotify playlist.
  • The user inputs a date (in the format yyyy-mm-dd).
  • The script extracts the top 100 songs played on that date using Beautiful Soup.
  • Finally, the songs are added to a new playlist on Spotify.
