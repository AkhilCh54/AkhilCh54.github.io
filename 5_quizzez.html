Quizzez (GUI and API)

Quizzes are a popular form of entertainment and education. They are used in schools, businesses, and even on social media. Quizzes can be a fun way to test your knowledge, learn new things, and compete with friends.

I created a version of it using Python. Here are some of the highlights:

  • Built using Object-Oriented Programming and integrated a GUI and API.
  • Used Python's Tkinter module for the graphical interface.
  • The quiz is structured as True/False questions.
  • API fetches questions from the Open Trivia Database.
  • Two buttons allow user interaction. If the answer is correct, the score increases.
  • Each session consists of 10 questions.
